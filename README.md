# esx_airplanedealer
## How to install
• Import `esx_aerodealer.sql` to your database
• Donwload the script and drop it to your `ressources/` folder
• Add `ensure esx_airplanedealer` to your `server.cfg`
• Configure config.lua
• Enjoy !
